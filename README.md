Простое одностраничное веб-приложение, позволяющее просматривать календарь, переключаться между месяцами и сохранять текстовые заметки на конкретные даты. Проект разработан с нуля на чистых веб-технологиях без использования сторонних библиотек, фреймворков или серверной части.

Запуск проекта:
1. Скачайте архив с файлами
2. Разархивируйте и откройте `index.html` в любом браузере (Chrome, Yandex, Firefox и др.)

Данный проект создан **исключительно в учебных целях** для сдачи, по дисциплине «Введение в профессиональную деятельность».
